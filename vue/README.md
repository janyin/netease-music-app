# 🎵 仿网易云音乐移动端网页【Vue 实现】

## 🍓 预览

## 😯 技术

- 构建工具：Vue cli3
- 路由，数据管理：Vue router + Vuex
- 数据请求：axios
- 歌词滚动：better-scroll
- UI 库：Vux

## 🏃 运行

1、clone

```git
git clone https://github.com.cnpmjs.org/janyin/vue-music-app.git

cd vue-music-app
```

2、安装依赖

```git
cd vue

yarn
```

3、本地运行

```git
yarn serve
```

4、打包

```git
yarn build
```

## ⬆️ 功能

- 推荐歌单
- 歌单详情
- 歌曲搜索
- 歌曲排行榜
- 歌词滚动
