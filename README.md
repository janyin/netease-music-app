# 网易云音乐web app

  使用vue cli3构建的SPA移动端网页,有搜索、播放、和歌单功能 <br>
  api来源：https://github.com/Binaryify/NeteaseCloudMusicApi
  整体UI就是网易云音乐官网的，播放就一个组件没写单独页面，用的是原生的H5<audio>
