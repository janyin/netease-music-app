## 🏃 运行

以 next 框架为例

1、clone

```git
git clone https://github.com.cnpmjs.org/janyin/vue-music-app.git

cd vue-music-app
```

2、安装依赖

```git
cd next

yarn
```

3、本地运行

```git
yarn dev
```

4、打包

```git
yarn build
```
