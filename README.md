功能页面都比较简单，但是常见需求都有(搜索、列表、页面跳转、数据状态管理)

适合拿来练手的入门级前端 Demo

## 🏃 运行

以 next 框架为例

1、clone

```git
git clone https://github.com/janyin/netease-music-app

cd netease-music-app
```

2、安装依赖

```git
cd next

yarn
```

3、本地运行

```git
yarn dev
```

4、打包

```git
yarn build
```

## ⬆️ 功能

- 推荐歌单
- 歌单详情
- 歌曲搜索
- 歌词滚动播放
